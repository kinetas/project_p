# 데이터베이스 1차 완성본 정리 (원본 셀 기준)

`데이터베이스-1차 완성본.drawio`의 **실제 테이블 셀(value) 값**을 그대로 옮긴 문서입니다.  
추정/자동생성 없이 컬럼명, 타입, 키 표기를 원본 그대로 반영했습니다.

## 테이블 정의

### `MARKET_INDEX`

- `BIGINT id (PK)`
- `DATE basDd`
- `VARCHAR idxNm`
- `Double flucRt`
- `Double opnprcIdx`
- `Double clsprcIdx`
- `Double cmpprevddIdx`
- `BIGINT mktcap`

### `EXCHANGE`

- `VARCHAR curUnit (PK)`
- `DATE baseDate`
- `VARCHAR country`
- `Double dealBasR`
- `DOUBLE changeRate`
- `Double changeAmount`

### `COMPANY`

- `VARCHAR stock_code (PK)`
- `VARCHAR corp_code (UK)`
- `VARCHAR corp_name`
- `CHAR corp_cls`
- `DATETIME created_at`
- `DATETIME updated_at`

### `FINANCIAL_STATEMENT`

- `BIGINT id (PK)`
- `CHAR bsns_year (UK)`
- `VARCHAR stock_code (FK, UK)`
- `VARCHAR reprt_code (UK)`
- `VARCHAR fs_div (UK)`
- `BIGINT revenue`
- `BIGINT operating_income`
- `BIGINT net_income`
- `BIGINT total_assets`
- `BIGINT total_liabilities`
- `BIGINT total_equity`

### `STOCK_PRICE`

- `VARCHAR srtn_cd (PK, FK)`
- `DATE bas_dt (PK)`
- `BIGINT clpr`
- `BIGINT lstg_st_cnt`
- `BIGINT mrkt_tot_amt`
- `DATETIME created_at`
- `DATETIME updated_at`
- `BIGINT mkp`
- `DOUBLE flt_rt`

### `DIVIDEND_INFO`

- `VARCHAR corp_code (PK, FK)`
- `VARCHAR dividend_kind`
- `BIGINT dividend_amount`
- `DATETIME stlm_dt`

### `STOCK_INDICATOR`

- `VARCHAR stock_code (PK, FK)`
- `DOUBLE per`
- `DOUBLE pbr`
- `DOUBLE roe`
- `DOUBLE debt_ratio`
- `DOUBLE dividend_yield`
- `DATETIME calculated_at`
- `DOUBLE eps`
- `DOUBLE bps`

### `TOP100`

- `DATE base_dt (PK)`
- `VARCHAR corp_code (FK)`
- `VARCHAR stock_code (FK, PK)`
- `INT score`

### `USER`

- `VARCHAR id(API.email) (PK)`
- `VARCHAR password`
- `VARCHAR nickname`
- `ENUM role`
- `DATETIME created_at`
- `DATETIME updated_at`

### `USER_FAVORITE`

- `VARCHAR user_id (PK, FK)`
- `VARCHAR stock_code (PK, FK)`
- `DATETIME created_at`
- `DATETIME updated_at`

### `INVESTMENT_JOURNAL`

- `BIGINT id (PK)`
- `VARCHAR user_id (FK)`
- `VARCHAR title`
- `TEXT content`
- `DATETIME created_at`
- `DATETIME updated_at`

### `COMMENT`

- `BIGINT id (PK)`
- `BIGINT journal_id (FK)`
- `VARCHAR user_id (FK)`
- `TEXT content`
- `DATETIME created_at`
- `DATETIME updated_at`

## 관계(원본 edge label 기준)

- `COMPANY -> FINANCIAL_STATEMENT` : `"stock_code"`
- `COMPANY -> STOCK_PRICE` : `"stock_code=srtn_cd"`
- `COMPANY -> DIVIDEND_INFO` : `"corp_code"`
- `COMPANY -> STOCK_INDICATOR` : `"stock_code"`
- `COMPANY -> TOP100` : `"stock_code"`
- `COMPANY -> USER_FAVORITE` : `"stock_code"`
- `USER -> USER_FAVORITE` : `"user_id"`
- `USER -> INVESTMENT_JOURNAL` : `"user_id"`
- `USER -> COMMENT` : `"user_id"`
- `INVESTMENT_JOURNAL -> COMMENT` : `"journal_id"`
